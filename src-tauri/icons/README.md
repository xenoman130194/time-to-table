# Иконки для приложения

Поместите в эту папку следующие файлы иконок:

- `32x32.png` - 32x32 пикселей
- `128x128.png` - 128x128 пикселей  
- `128x128@2x.png` - 256x256 пикселей (Retina)
- `icon.icns` - macOS иконка
- `icon.ico` - Windows иконка

## Генерация иконок

Вы можете использовать команду Tauri для автоматической генерации всех форматов:

```bash
npm run tauri icon путь/к/исходной/иконке.png
```

Исходная иконка должна быть минимум 1024x1024 пикселей в формате PNG.
